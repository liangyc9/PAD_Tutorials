# 跑步訓練課表：跑三天 + 囚徒健身兩天

## 課表總覽

- **訓練週期**：4 週（銜接 → 推進 → 刺激 → 整合）
- **每週結構**：Day 1（Zone 2 跑 + 力量 A）、Day 2（速度訓練）、Day 3（長跑）+ 力量 B（週二或週六）
- **目標賽事**：半馬準備（鋪路階段）
- **核心指標**：HR（心率）、配速（min/km）

---

---

## 設計邏輯

1. **Tempo 優先拉長時間**：Tempo 能力已穩定，因此以延長持續時間為主要目標
2. **Interval 採 4 分鐘以上**：避免心率 lag 造成訓練失效
3. **Long Run 漸進到 100 分**：系統性為半馬鋪路
4. **不追求 HR 160 以上爆衝**：保護有氧基礎，避免過度訓練
